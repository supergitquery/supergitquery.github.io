bytes

数据保存到硬盘,硬盘只能存储2进制,在我们常使用的文本中,文本会先以一种编码方式
转为十六进制,再转为2进制储存.而其他的类型文件都有自己的格式如jpg,png,mp4,mp3,
这些文件也是以二进制储存在硬盘上.

bytes类型,以16进制形式表示,2个16进制构成一个bytes也就是一个字节,以b’’来标识.
