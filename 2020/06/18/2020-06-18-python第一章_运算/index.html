运算

计算机可以进行的运算有很多种,运算按种类可以分为算数运算,比较运算,逻辑运算,赋值运算,成员运算,身份运算,位运算

算数运算

假设a=10,b=20

运算符 描述 实例
+ 加 - 两个对象相加 a + b 输出结果30
- 减 - 得到符数或是一个数减去另一个数 a - b 输出结果10
* 乘 - 两个数相乘或是返回一个被重复若干次的字符 a * b 输出结果200
/ 除 - x除以y b / a 输出结果2
% 取模 - 返回除法的余数 b % a 输出结果0
** 幂 - 返回x的y次幂 a**b 输出结果100000000000000000000
// 取整数 - 返回上的整数 b // a 输出结果2

比较运算

假设a=10,b=20

运算符 描述 实例
== 等于 - 比较两个对象是否相等 (a == b)返回False
!= 不等于 -比较两个对象是否不相等 a != b 返回False
> 大于 - 返回x是否大于y a > b 返回False
< 小于 - 返回x是否小于y a < b 返回True
>= 大于等于 - 返回x是否大于等于y a >= b 返回False
<= 小于等于 - 返回x是否小于等于y a <= b 返回True

逻辑运算

假设a=10,b=20

运算符 描述 实例
and 判断多个条件为真时,返回真 a>10 and b>10 返回False
or 判断任意条件为真时,返回真 a>10 or b>10 返回True
not 取反 not a>b 返回True

赋值运算

假设a=10,b=20

运算符 描述 实例
= 简单的赋值运算 c = a + b就是将a+b的结果赋值给c
+= 加法赋值运算符 c += a 等效于 c = c + a
-+ 减法赋值运算 c -= a 等效于 c = c - a
*= 乘法赋值运算 c *= a 等效于 c = c * a
/= 除法赋值运算 c /= a 等效于 c = c / a
%= 取模法赋值运算 c %= a 等效于 c = c % a
**= 幂赋值运算 c **= a 等效于 c = c ** a
//= 取整赋值运算 c //= a 等效于 c = c // a
