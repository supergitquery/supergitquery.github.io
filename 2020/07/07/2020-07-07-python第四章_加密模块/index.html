hashlib

MD5

讯息摘要演算法,一种被广泛使用的密码杂凑函数,可以产生一个128位的散列值.

输入任意长度的信息,经过处理,输出为128位的信息(数字指纹);
不同的输入得到的不同的结果(唯一性).

特点:
压缩性:任意长度算出的md5的长度都是固定的.
容易计算:从原数据计算出md5值很容易.
抗修改性:对原数据进行任何改动,修改一个字节生成的md5区别也会很大.
强抗碰撞:已知原数据和md5,想找到一个相同的md5值的数据是非常困难的.

用途:
数据加密
防篡改
防抵赖

SHA

安全哈希算法,主要适用于数字签名标准里面定义的数字签名算法,对于长度小于2^64位的消息,
SHA1会产生一个160位的消息摘要.当接受消息的时候,这个消息摘要可以用来验证数据的完整性.

后续由推出了SHA224 SHA256 SHA384 SHA512,生成的位数越长,
破解难度越大,同时需要的时间也就越长,目前流行的是SHA256.
